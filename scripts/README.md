### Scripts
## Overview 
This folder contains the scripts to set up the environment for the app on the raspberry pi

It contains:
* **setup_script.sh** used for installing the necessary librairies on the raspberry pi.

## Prerequisite
Before using the scripts in this section you may need to make them executable. For this simply run the following line:
```bash
sudo chmod +x ./scriptName
```

## Running the Scripts:
To run the scripts run:
```bash
sudo ./scriptName
```
